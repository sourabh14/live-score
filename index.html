live-score

live-score contains python script, to fetch live cricket match scores and show it as a notification on desktop screen. It supports user defined settings for notification popup, such as after every interval, after runs are scored, and after wickets fall.

Prerequisites

-Linux OS -python 2.7.6 and above

Installation

Download repo zip file from github page : https://github.com/sourabh14/live-score and extract.

Usage

Go to the script location $ cd "/path-to-live-score/src/"

run the script using $ python live-score
